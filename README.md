# Kayschima Dice

Eine moderne Flutter-App zum Würfeln mit frei wählbarer Würfelanzahl und einfacher Auswahl einzelner
Würfel.

## App-Inhalte

- Würfeln mit 1 bis 10 Würfeln
- Sichtbarer Wurfzähler (Anzahl der bisherigen Würfe)
- Würfel nach dem ersten Wurf antippen, um sie zu sperren/entsperren
- Zufallswurf nur für nicht gesperrte Würfel
- Reset-Funktion, um die Runde vollständig zurückzusetzen
- Klare, mobile-optimierte Oberfläche mit Material Design

## Schnellstart

Voraussetzungen:

- Installiertes Flutter SDK
- Ein konfiguriertes Zielgerät (Android Emulator, iOS Simulator oder physisches Gerät)

Projekt lokal starten:

```bash
flutter pub get
flutter run
```
